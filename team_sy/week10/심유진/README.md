# **Must**

### **공통**
   - [O] [뱅크샐러드 홈페이지]([https://www.banksalad.com/](https://www.banksalad.com/))에 보이는 모든 요소를 **구현**해야 한다.
   - [O] **시멘틱 태그**를 적절히 사용해야 한다.
   - [O] **웹 폰트**로 “Pretendard”를 적용해야 한다.
   - [O] **이미지**는 뱅크샐러드 홈페이지의 리소스를 사용한다.
